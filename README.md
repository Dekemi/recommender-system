## 🔗 Live Demo

👉 https://huggingface.co/spaces/dekemi/Recommender-App

---

## 📌 Project Overview

This project implements a **collaborative filtering recommendation system** using user-product interactions.

It suggests products based on **user similarity using cosine similarity**.

---

## ⚙️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit

---

## 🧠 Features

* User-item matrix creation
* Cosine similarity-based recommendations
* Precision@K evaluation metric
* Interactive UI using Streamlit

---

## 📊 How It Works

1. Convert user data into a matrix
2. Compute similarity between users
3. Recommend unseen products
4. Evaluate using Precision@K

---

## 🚀 Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
